Android Studio超级详细安装教程（intel/英特尔）

本教程适合CPU为intel/英特尔的计算机使用，如果你的CPU为AMD请移步至Android Studio超级详细安装教程（AMD）

查看计算机CPU方法：右击此电脑—>属性

前言

我的Android Studio的安装过程和老师的有些许不同，因为我们没有现成的Android文件，所以首先我们只能自己去创建。本教程亲测可以成功运行并输出Hello World！

本教程现在只是初稿，后期会根据大家的反馈不断地进行修改和更新！也欢迎大家纠错，共同提升本教程的质量！

Android Studio（intel/英特尔）安装教程

  1. 安装包下载：点击前往资源区寻找Android Studio安装包，下载之后并打开

  2. 接下来按照如下步骤安装

    勾选Android Virtual Device，点击Next

    点击Install之后，等待安装完成

    勾选Start Android Studio，点击Finish运行

    选择Do not import settings，点ok

    选择Standard安装方式并，点击Next

    选择主题背景，点击Next

    接下来就是漫长的等待，等待的时间取决于你的网络环境

    完成之后点击Finish

  3. 创建一个新的Hello World程序！

    选择Empty Activity，点击Next，这里会卡住一会，不要慌等一等

    按自己的需求去修改，改完后点击finish

    然后等待项目构建完成。这里由于是第一次启动，所以需要下载gradle以及Android项目需要引用的包，等待的时间取决于你的网络环境

    中途会弹出如下图，点击允许访问

    在等待项目构建的同时，我们去下载模拟器，点击如下图的位置

    首先把①处的两个复选框勾选上，稍微等待一会，等出现数据之后，下滑找到图中的②处并勾选上，然后点击ok，下一个界面再次点击ok

    然后选Accept，点击Next，然后就静静地等待下载完毕吧，下载完成之后点finish即可

  4. 接下来去运行第一个Hello World！程序

    点击如图位置，然后点击Create Virtual Device…

    下滑找到并选中如下图框内，点击Next

    找到下图①处并点击，然后点击Next

    右下角出现错误提示不用管它

    之后在任务栏出现了一个新任务，打开它，会出现一个小窗口如图。接下来点击那个绿色的小三角，留意这个小窗口

    看到界面出现Hello World！便成功了！

安装中可能遇到的问题

  1. android studio虚拟机无法启动 The system requirements are not satisfied、vmx enabled no、vmx supported no

    在任务管理器 -> 性能 -> 右下角，看看虚拟化是 已禁用 还是 已启用,若是已禁用(或者使用haxm setup安装程序时报错system requirements are not satisfied 、 vmx enabled no、vmx supported no )

    如果显示禁用，则需要

    • 进入bios界面，将Config -> CPU -> Intel Virtualization Technology 设为Enabled，重启电脑，再去任务管理器中查看是否已启用。（部分主板还需要将 Security -> Virtualization -> Intel® Virtualization Technology 设为Enabled，最好检查一下有无这个设置，有的话顺便设置好）

    • 进入BIOS模式有很多种方式，因为主板不同，可能按的按键也略有差异。一般是按的是F1或F2，要根据自己的硬件品牌确认（百度搜）。

  2. 安装过程中出现Your SDK location contains non-ASCII characters.错误，如下图

    问题原因：

    • 安装路径出现了中文字符

    解决办法：

    • 换个路径，路径中不要出现中文

  3. 出现如下图问题，即gradle安装失败

    解决办法：

    • 首先尝试点击上方的Try Again重新下载，如下图

    • 如果仍然报错，需要手动下载gradle，需要下载指定版本即gradle-5.4.1-all，这里我提供了下载渠道

      1. 百度网盘下载：https://pan.baidu.com/s/13TdU7QuCx2VMv1NeXCkMwg 提取码：11q2
      2. 加我QQ发送（备注好目的）

    • 下载完之后打开C:\Users\你的用户名\.gradle\wrapper\dists将里面的gradle-5.4.1-all删掉，然后把刚刚下载下来的压缩包解压到此处，然后重启Android Studio等待加载完毕即可。
